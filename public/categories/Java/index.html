    Java란?#

    Java는 객체지향 프로그래밍 언어로, 플랫폼 독립적인 특성을 가지고 있습니다.

    JVM#

    Java Virtual Machine은 Java 바이트코드를 실행하는 가상 머신입니다.

    JAVA
    1public class Main {
2    public static void main(String[] args) {
3        System.out.println("Hello, Java!");
4    }
5}

    기본 자료형#

    자료형 크기 범위
    int 4바이트 -2^31 ~ 2^31-1
    long 8바이트 -2^63 ~ 2^63-1
    double 8바이트 부동소수점

    Java Stream API 활용 가이드

    February 28, 2026
    Development, Java
    Java, Stream API, Functional Programming, Programming

    Stream API란?#

    Java 8에서 도입된 Stream API는 컬렉션 데이터를 선언적으로 처리할 수 있게 해줍니다.

    중간 연산#

    JAVA
    1List<String> names = members.stream()
2    .filter(m -> m.getAge() > 20)       // 필터링
3    .map(Member::getName)                // 변환
4    .sorted()                            // 정렬
5    .distinct()                          // 중복 제거
6    .collect(Collectors.toList());

    최종 연산#

    JAVA
    1// 개수
2long count = members.stream().filter(m -> m.getAge() > 20).count();
3
4// 합계
5int totalAge = members.stream().mapToInt(Member::getAge).sum();
6
7// 그룹핑
8Map<String, List<Member>> byTeam = members.stream()
9    .collect(Collectors.groupingBy(Member::getTeamName));

    flatMap 활용#

    JAVA
    1List<String> allTags = posts.stream()
2    .flatMap(post -> post.getTags().stream())
3    .distinct()
4    .collect(Collectors.toList());

    주의사항#

    • Stream은 한 번만 사용 가능 (재사용 불가)
    • 병렬 스트림(parallelStream())은 신중하게 사용
    • 무한 스트림 사용 시 limit() 필수

    정리#

    Stream API를 활용하면 코드의 가독성과 표현력을 크게 향상시킬 수 있습니다.
