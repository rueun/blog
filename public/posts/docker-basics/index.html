    Docker 기초 - 컨테이너의 개념과 기본 명령어

    2026. 02. 15. 00:00 · 읽는데 1분

    Docker란?#

    Docker는 애플리케이션을 컨테이너라는 격리된 환경에서 실행할 수 있게 해주는 플랫폼입니다.

    컨테이너 vs 가상머신#

    구분 컨테이너 가상머신
    격리 수준 프로세스 수준 OS 수준
    시작 시간 초 단위 분 단위
    리소스 사용 가벼움 무거움

    기본 명령어#

    BASH
     1# 이미지 다운로드
 2docker pull nginx:latest
 3
 4# 컨테이너 실행
 5docker run -d -p 8080:80 --name my-nginx nginx:latest
 6
 7# 컨테이너 목록 확인
 8docker ps
 9
10# 컨테이너 중지 및 삭제
11docker stop my-nginx
12docker rm my-nginx

    Dockerfile 작성#

    DOCKERFILE
    1FROM openjdk:17-jdk-slim
2COPY build/libs/app.jar app.jar
3ENTRYPOINT ["java", "-jar", "app.jar"]

    Docker를 활용하면 개발 환경과 운영 환경의 차이를 최소화할 수 있습니다.
