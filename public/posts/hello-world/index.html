    소개#

    안녕하세요! 이 블로그는 Hugo와 GitHub Pages를 사용하여 만들었습니다.

    Hugo란?#

    Hugo는 Go 언어로 만들어진 정적 사이트 생성기입니다. 빠른 빌드 속도와 풍부한 테마가 장점입니다.

    주요 특징#

    • 빠른 빌드 속도
    • 다양한 테마 지원
    • Markdown 기반 콘텐츠 작성
    • 간편한 배포

    GitHub Pages#

    GitHub Pages를 사용하면 무료로 정적 웹사이트를 호스팅할 수 있습니다.

    설정 방법#

    1. GitHub 레포지토리 생성
    2. Hugo 프로젝트 초기화
    3. GitHub Actions 설정
    4. Push 후 자동 배포
